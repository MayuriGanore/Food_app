## Run Locally

Clone the project

```bash
  git clone https://github.com/MayuriGanore/Food_app
```

Go to the project directory

```bash
  cd Food_app
```



Start the server

```bash
  npm run server
```
Start the admin

```bash
  npm run dev
  (go to /admin/vite-project)
```

Start the frontend

```bash
  npm run dev
```
